# Privacy

Trusted Research sends user content to the backend service you configure.

## Data that may be sent

- user query
- current selection
- current note body
- current note title
- current note path
- account email and password for Control Plane login

## Storage behavior

- access token: memory only for the current plugin session
- refresh token: memory only for the current plugin session
- Control Plane email and password: stored in plugin settings when you configure them

## What this repo does not provide

- a hosted backend
- a shared privacy policy for third-party runtime providers

You are responsible for operating or choosing a backend that matches your privacy requirements.
